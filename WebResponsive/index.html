Bienvenido al sitio web de José Manuel Palacios López

Este espacio ha sido diseñado para la práctica de la asignatura de DIW. Este módulo forma parte del currículo de 2º año del CFGS Desarrollo de Aplicaciones Web. El ciclo posibilita al estudiante los conocimientos para acceder al mercado como desarrollador y programador de aplicaciones en entornos web.
