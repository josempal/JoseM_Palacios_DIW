Bienvenido al sitio web de Jośe Manuel Palacios López. Este espacio ha sido diseñado para la práctica de la asignatura de DIW. Este módulo forma parte del currículo de 2º año del CFGS Desarrollo de Aplicaciones Web. El ciclo posibilita al estudiante los conocimientos para acceder al mercado como desarrollador y programador de aplicaciones en entornos web.


El centro del que procede la petición de creación de esta web es el I.E.S. Alixar (Castilleja de la Cuesta, Sevilla). Este es el listado de módulos de dicho ciclo formativo para el curso académico 2020/2021:
